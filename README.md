# XDR Platform Comparison

Compare Taegis XDR and competitor cybersecurity platforms.

## Run locally

```bash
npm install
npm run dev
```

**Use the URL shown in the terminal.** The dev server usually starts at http://localhost:3000, but if that port is already in use it will use the next one (e.g. http://localhost:3001). Open the URL that appears after "Local:" in the terminal so the app loads correctly.

## Build

```bash
npm run build
npm run start
```

Production server defaults to port 3000 unless the port is in use.
